Fonds de page pour les rapports PDF
====================================

Pour que le fond de page s'affiche dans les PDF, placez les images suivantes dans ce dossier :

  - fond_portrait_a4.png   (note de prise en charge, format portrait A4 595x842 pt)
  - fond_paysage_a4.png    (listing des PEC, format paysage A4 842x595 pt)

Sans ces fichiers, le bandeau "background" des rapports restera vide (aucune image de fond).
